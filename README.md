# Jin Yu / 진 — Portfolio

Personal portfolio site for Jin Yu, journalist, copywriter, and communications professional.

## How to publish on GitHub Pages

1. Create a new repository on GitHub named `username.github.io` (replace `username` with your GitHub username)
2. Upload all files from this folder into the repository — keep the folder structure exactly as is
3. Go to **Settings → Pages** in your repository
4. Under **Source**, select **Deploy from a branch** → **main** → **/ (root)**
5. Click Save — your site will be live at `https://username.github.io` within a few minutes
